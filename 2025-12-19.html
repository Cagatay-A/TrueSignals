Sistem Bilgisi

Bu tablo GitHub Actions tarafından her 5 dakikada bir güncellenir.

• Yeşil renk AL emirlerini veya kar durumunu gösterir

• Kırmızı renk SAT emirlerini veya zarar durumunu gösterir

• Veriler otomatik olarak her 30 saniyede bir yenilenir

• Veri Akışı: Local API → GitHub Actions → GitHub Pages JSON → Bu Sayfa

• Çevrimdışı durumda cache verileri gösterilir
