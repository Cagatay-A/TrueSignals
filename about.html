TrueSignal Hakkında

Amacımız

TrueSignal, GitHub Actions ve GitHub Pages kullanarak gerçek zamanlı emir takip sistemi oluşturan bir projedir.

Kullanılan Teknolojiler

  • GitHub Pages: Ücretsiz ve güvenli hosting
  • GitHub Actions: Otomatik veri senkronizasyonu
  • Local API: Gerçek zamanlı emir verileri
  • HTML5/CSS3/JavaScript: Modern web teknolojileri
  • Bootstrap 5: Responsive tasarım framework'ü

Temiz URL Sistemi

Bu projede, kullanıcıların .html uzantısı görmeden temiz URL'ler kullanmasını sağlıyoruz:

https://cagatay-a.github.io/TrueSignals/about

Yukarıdaki URL otomatik olarak about.html dosyasını açar.

Veri Akışı

  1. Local API (192.168.1.3:3000): Gerçek emir verileri
  2. GitHub Actions: Her 5 dakikada bir veri çeker
  3. GitHub Pages JSON: Verileri JSON formatında depolar
  4. Web Arayüzü: JSON'dan verileri okur ve gösterir
  5. Cache: Çevrimdışı durumda cache verilerini gösterir

Güvenlik Özellikleri

  • HTTPS güvenli bağlantı
  • GitHub Pages güvenli altyapısı
  • Local API'ye doğrudan erişim yok
  • JSON doğrulama ve hata yönetimi
  • Cache mekanizması ile çevrimdışı çalışma

Responsive Tasarım

Tüm sayfalar mobil, tablet ve masaüstü cihazlarda optimize edilmiştir.

  • Mobil uyumlu navigation
  • Responsive tablolar
  • Adaptif grid sistem
  • Touch-friendly butonlar
